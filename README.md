# CodeAlpha Data Redundancy Removal System

## Description

This project was developed as part of the CodeAlpha Cloud Computing Internship. It is a Python Flask web application that identifies and prevents duplicate data entries by validating new records before storing them in the database.

## Features

- Add New Records
- Detect Duplicate Data
- Prevent Redundant Entries
- Store Only Unique Data
- SQLite Database Integration
- User-Friendly Web Interface

## Technologies Used

- Python 3
- Flask
- HTML5
- CSS3
- SQLite
- Git & GitHub
- Kali Linux

## Installation

Install Flask:

```bash
sudo apt install python3-flask -y
```

or

```bash
pip install Flask
```

## Run the Project

```bash
python3 app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
